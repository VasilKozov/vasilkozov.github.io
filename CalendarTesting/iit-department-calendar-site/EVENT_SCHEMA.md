# Event Field Schema

This file documents the fields used in `data/events.xlsx`.

The website currently reads every worksheet whose name is a four-digit year. It expects the first row to contain column names.

## Required fields for a useful public event

- `id`
- `date`
- `title_bg`
- `category`
- `description_bg`
- `facebook_url` or another evidence link
- `report_notes`

## Recommended category list

- Academic event
- Scientific activity
- Student activity
- Student achievement
- Student outreach
- Industry collaboration
- International collaboration
- Guest lecture
- Public visibility
- Outreach/school collaboration
- Scientific/outreach activity
- Student information

## Recommended icon list

- `rocket` - new initiative / student event
- `magnifier` - science / research / analysis
- `lock` - internal or strategic activity
- `envelope` - communication / publication / announcement
- `gear` - technical or industry activity
- `star` - achievement / milestone

## Recommended color list

- `blue`
- `blue-dark`
- `teal`
- `orange`
- `red`
- `green`
- `green-dark`
